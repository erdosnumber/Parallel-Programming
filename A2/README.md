# Running the Code

To run the code, follow these steps:

1. Compile all executables using the command `make all`. Note that this will generate the executables in the same directory.
2. Once compiled, preprocessed images can be found in the folder `pre-proc-img/`, and the outputs will be written to a folder `output/`. In subtasks 3 and 4, the code is hardcoded to read 10,000 files in the `pre-proc-img/` folder.
